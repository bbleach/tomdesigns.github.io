CSV exporting with python

Revisiting the first of three summer assignments in python to add CSV exporting.
Quick link to code

Overview

This is identical to the average speed program, although it now exports data to a CSV file. The breakdown of this project will talk only about the file handling.

Breakdown

Use of libraries

The CSV library was used

import os
import csv

Creating the file

When the program begins, it checks to see if a 'db.csv' file exists, and creates one if nescessary.

    if (os.path.isfile('db.csv')):
        pass
    else:
        with open('db.csv', 'w', newline='') as f:
            database = csv.DictWriter(f, fieldnames=['Registration', 'Enter', 'Exit', 'Time in zone', 'Speed', 'Over limit?'])
            database.writeheader()

Adding rows of data

By opening the file with the 'a' flag as opposed to 'w,' each row is appended to the existing file. Each field is filled with the same variables used throughout the calculations within the code.

  with open('db.csv', 'a', newline='') as f:
        database = csv.DictWriter(f, fieldnames=['Registration', 'Enter', 'Exit', 'Time in zone', 'Speed', 'Over limit?'])
        database.writerow({'Registration' : carReg, 'Enter' : t1, 'Exit' : t2, 'Time in zone' : td, 'Speed' : speed, 'Over limit?' : limit})

Average Speed Camera Program

The first of three summer assignments in python. (Also my first python program)
Quick link to code

Task introduction

On many major roads average speed checks are in place. Two sensors are placed a known distance apart and vehicle number plate recognition is used to identify a vehicle and the time it enters the section of road being monitored. The time is recorded when the vehicle leaves the monitored section. By using the time taken to travel the known distance, the average speed of a vehicle can be calculated. Write a program for calculating average speeds for a vehicle travelling through a section of road.

Overview

When the program is run, the user is prompted to enter a registration plate. The program will attempt to confirm that this number plate conforms to the default UK format. Then, the user is promted to enter when the car enters and exits the average speed zone, along with the speed limit. After this, the program uses speed = distance / time to calculte the average speed of the car before deciding whether it breaks or abides by the speed limit.

Validating an input

Regular expressions allowed me to check that the input "carReg" matched a format. This was achieved as follows:

A re.match statement checks for two letters, two numbers, a space and then 3 letters.

while True:
    carReg = input("Enter car registration: \n")
    if re.match ('^[A-Za-z]{2}[0-9]{2}[ ][A-Za-z]{3}$', carReg):
        carReg = carReg.upper()
        break
    else:
        print("Invalid registration, please use the form AA11 AAA (Where A is any letter A-Z, and 1 is any integer < 10.")

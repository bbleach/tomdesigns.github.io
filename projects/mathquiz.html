Math Quiz Program

The third of three summer assignments in python.
Quick link to code

Task introduction

A primary school teacher wants a computer program to test the basic arithmetic skills of her students. The program should generate a quiz consisting of a series of random questions, using in each case any two numbers and addition, subtraction and multiplication. The system should ask the studentâ€™s name, then ask 10 questions, output if the answer to each question is correct or not and produce a final score out of 10.

Overview

As the program is run, the user is prompted to enter a name. Then, 2 random integers and a random operator are generated to provide a random question. The user is prompted for an answer is a score is kept.

Breakdown

Use of libraries

Random, Time and OS were used in the program.

import random
import time
import os

Generating random integers and operators

Generating the random numbers was simple, int1 and int2 are assigned a number each between 1 and 10.

int1 = random.randint(1,10)
int2 = random.randint(1,10)

To generate a random operator, I generated a random number between 1 and 3 and used elif statements to assign each a different operator.

oprt = random.randint(1,3)

if oprt == 1:
    op = "+"
    answer = (int1+int2)
elif oprt == 2:
    op = "-"
    answer = (int1-int2)
elif oprt == 3:
    op = "*"
    answer = (int1*int2)

At this stage, I also generated the answer to compare the user input to.

Verifying the answer

An if statement was used to check the answer. If it is correct, 1 is added to the score.

if int(ans) == int(answer):
    print("Correct!")
    score +=1
    print("Current score: {} \n".format(score))
else:
    print("Incorrect, the correct answer is: {} \n".format(answer))
    print("Current score: {} \n".format(score))

.total_seconds() allowed me to convert the time delta object into an integer value, which is then converted into hours as a float value.

Managing structure/flow

As the program begins, the variables score and question are set to 0 and 1 respectively. This means that each question is numbered, and the score can be seen throughout.

The question generating code uses a for loop in order to repeat 10 times.

for x in range(0,10):

After each question, 1 is added to the question number with question +=1 and, granted it is correct, 1 is added to the score with score +=1

Also, after each question, time.sleep(3) allows the user to review their current score before the console is cleared with os.system('cls')
